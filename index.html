Aprendiendo a mejorar el aspecto del HTML con Css

En HTML, puedes crear listas con viñetas (también conocidas como listas con viñetas o listas desordenadas) utilizando la etiqueta < ul> (unordered list) y los elementos < li> (list item) para cada elemento de la lista. Puedes personalizar estas viñetas utilizando CSS para darles diferentes estilos. Aquí hay un ejemplo de cómo hacerlo: Para utilizar viñetas en HTML debes usar la etiqueta ul y dentro de la etiqueta ul agregar las etiquetas li
